# coc-rescript

A coc.nvim language server for rescript.

## Install

`:CocInstall coc-rescript`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-rescript-keymap)`

## Lists

`:CocList demo_list`
